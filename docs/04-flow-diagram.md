## 关键区别

|             | 正常结算                        | 争议处理                                                    |
|-------------|-----------------------------|---------------------------------------------------------|
| 分叉点         | liveness 内无质疑               | liveness 内有质疑                                           |
| requestTime | 全程 T1                       | 质疑后更新为 T2                                               |
| OO 状态流转     | Requested→Proposed→Resolved | Requested→Proposed→Disputed→Requested→Proposed→Resolved |
| 结算结果        | YES 赢                       | NO 赢                                                    |
| Bond 结局     | Proposer 取回                 | Proposer 损失，Disputer +100                               |
| User1 收益    | +1000 USDC                  | 0                                                       |
| User2 收益    | 0                           | +2000 USDC（赎回）+100（DVM）                                 |
